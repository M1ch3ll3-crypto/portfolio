4. CGPA Calculator

A tool for Miva students to calculate semester GPA instantly. Input your course units and grades, and it computes your GPA using the 5.0 scale. Built to help students track academic performance without Excel.

Workflow:
  1. User enters Course Code, Units, and Grade for each course
  2. JavaScript stores data in an array of objects
  3. On submit, function loops through array to calculate total grade points
  4. GPA = Total Grade Points / Total Units, displayed with 2 decimals
